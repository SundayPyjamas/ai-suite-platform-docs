## What is this?

The **Images** tool lets you create pictures by describing them in plain words. Type "a cozy coffee shop on a rainy evening, warm lights" and the AI draws it. You can make logos, illustrations, backgrounds, social media pictures, and more.

> It's like having an artist who paints whatever you describe — in seconds.

## How to make your first image

<Steps>
<Step title="Open Images">
  Click **Images** in the left menu. You'll find a box to describe your picture.

  <Frame caption="The Images playground. Describe your picture in the box, then generate.">
    <img src="/images/user-guide/images-playground.png" alt="The Images tool with a description box and a generate button" />
  </Frame>
</Step>
<Step title="Describe the picture">
  Type what you want to see. Be visual. For example:

  > *A friendly orange cat sitting on a stack of books, cartoon style, soft colors, white background.*
</Step>
<Step title="Generate">
  Click **Generate** and wait a few seconds while the AI draws it.

  <Frame caption="Your generated image appears, ready to download or refine.">
    <img src="/images/user-guide/images-result.png" alt="A generated image shown in the Images tool" />
  </Frame>
</Step>
<Step title="Download or try again">
  Like it? Download it. Want something different? Change your description and generate again.
</Step>
</Steps>

## How to describe a good image

The trick is to paint a picture with words. Include:

<CardGroup cols={2}>
  <Card title="The subject" icon="star">
    What's in it? *"a mountain lake", "a smiling robot".*
  </Card>
  <Card title="The style" icon="palette">
    How should it look? *"photo", "cartoon", "watercolor", "minimal".*
  </Card>
  <Card title="The mood / colors" icon="sun">
    *"warm and cozy", "bright and playful", "dark and moody".*
  </Card>
  <Card title="Extra details" icon="plus">
    *"white background", "top-down view", "soft lighting".*
  </Card>
</CardGroup>

## Editing an image

Some image tools let you **change an existing picture** — for example, "make the background blue" or "add a hat." Look for an edit option after you generate. Describe the change and the AI updates the picture.

<Tip>
If the first result isn't quite right, add more detail rather than starting over. "Make it brighter and add a blue sky" often gets you there faster.
</Tip>
