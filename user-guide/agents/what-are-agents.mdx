## What is an agent?

A normal [chat](/user-guide/first-chat) answers your questions. An **agent** goes further: it can *do things* — use tools, take several steps, create files, and work toward a goal.

> **Chat** is like asking a friend a question.
> An **agent** is like asking an assistant to *go and get something done* — and it takes the steps needed to finish.

## The Core Agent lives in the Agents tab

Click **Agents** in the left menu and you'll meet your **Core Agent** — the powerful helper that's built into every workspace and ready to go. You don't have to set anything up. Just open the Agents tab and start giving it tasks.

## What makes the Core Agent special

Unlike a plain chat, the Core Agent can:

  <Card title="Take multiple steps" icon="list-check">

  <Card title="Create files & artifacts" icon="file-lines">

We'll cover all of these in [What the Core Agent can do](/user-guide/agents/capabilities).

## Chat vs. the Core Agent, at a glance

| | Plain Chat | Core Agent |
|---|---|---|
| Answers questions | ✅ | ✅ |
| Remembers the conversation | ✅ | ✅ |
| Uses connected tools | — | ✅ |
| Takes multiple steps on its own | — | ✅ |
| Creates files & artifacts | — | ✅ |

If a plain chat does what you need, use [Chat](/user-guide/first-chat). Reach for the **Core Agent** when a task needs **tools**, **several steps**, or a **file** as the result.
